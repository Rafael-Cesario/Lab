##### Um aplicativo para salvar suas coisas favoritas.

## Feito com

    - GraphQL / Apollo-Server
    - NextJS
    - React
    - Typescript

## Funcionalidades

    - Adicionar algum conteudo como favorito.
    - Escolher uma nota para o favorito.
    - Escolher uma imagem como capa.
    - Adicionar a uma categoria.
    - Filtros.
