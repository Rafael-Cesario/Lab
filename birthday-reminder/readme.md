# BirthDay Reminder
Um simples app para anotar datas importantes e algumas anotações para facilitar a escolha de presentes ou surpresas.

## Feito com
    - webComponents
    - lit
    - typeScript
    - webPack
    - mongoDB
    - API REST.

## Funcionalidades
    - Adicionar Pessoas.
    - Adicionar datas.
    - Adicionar anotações para gostos, personalidades, presentes, notas.
